Ce module est un module de magasin qui a pour vocation de vendre les objets provenant d'autres jeux de rôles.

Todd peut être installé sur une partie en cours, et il attend les aventuriers à différents endroits suivant le jeu sur lequel il est installé :
- BGEE : à la Porte de Baldur dans les docks près de l'eau
- SOD : dans la taverne des 3 vieilles chopes
- BG2EE : à Athkatla dans les docks près de l'eau
- TOB : dans la taverne d'Amkethran
- IWDEE : dans l'auberge de Kuldahar.
- BGT : tous les emplacement cités sauf celui de IWDEE.
- EET : tous les emplacement cités sauf celui de IWDEE.
- Les Fosses Noires I & II : dans l'enclaves des marchands.

Pour l'instant il inclut les objets de 
- Morrowind (et ses extensions) 
- Arcanum

Le format de ce module le prédestine à s’agrandir : pour qu'un autre jeu voit ses objets ajoutés à l'inventaire de Todd, il faut 3 critères :
- JDR plus au moins médiéval.
- Base de donnée bien fournie quelques part sur le net ou en livre.
- Compatibilité 80% environ (exemple : Pillars of eternity ou Path of Exil ne sont pas compatibles, trop d'effets n'existant pas dans BG2 ou AD&D)
Si vous connaissez un jdr regroupant ses trois critères, proposez le moi, je l'ajouterais probablement au module.

La finalité de ce module serait de donner aux objets les vraies icônes telles qu'ils les ont dans le jeu original. Mais pour l'instant je butte sur les icônes d'Arcanum, qui si elle supporte bien la conversion au format BG2, supportent très mal la réduction au format 32x32.
